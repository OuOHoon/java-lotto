# 로또
로또를 구매하고 당첨 결과를 통계로 출력하는 프로그램

## 기능 목록
### Lotto 클래스
- 당첨 번호 입력이 올바른지 검증한다.
  - 당첨 번호의 숫자 범위는 1~45 사이다.
  - 당첨 번호의 숫자는 중복되지 않는다.
  - 당첨 번호는 중복되지 않는 숫자 6개다.
- 보너스 번호 입력이 올바른지 검증한다.
  - 보너스 번호는 당첨 번호와 중복되지 않는다.
- 로또 번호 조합을 생성한다.
- 당첨 번호를 입력받는다.
- 보너스 번호를 입력받는다.
- 당첨 번호와 로또 번호를 비교한다.
- 당첨 금액을 계산한다.
- 수익률을 계산한다.
- 당첨 통계를 출력한다.

### User 클래스
- 로또 구매 금액을 입력받는다.
- 로또 구매 금액이 올바른지 검증한다.
  - 구매 금액은 1,000 단위로 나누어 떨어져야 한다.
- 로또를 구매한다.

### LottoError 클래스
- 에러 메시지를 출력한다.
